# CELIA-014 - Edit and Delete Transactions

## Epic

Transactions

## Goal

Allow users to modify existing transactions.

## User Story

As a user,
I want to update or remove transactions
so that my records stay accurate.

## Scope

Support:

- Edit transaction
- Delete transaction
- Confirmation dialog

Reuse the Add Transaction modal.

## Acceptance Criteria

- [ ] Existing transaction can be edited
- [ ] Existing transaction can be deleted
- [ ] Confirmation displayed before deletion
- [ ] Changes persist
- [ ] Tests added
- [ ] Verification pipeline passes

## Dependencies

- CELIA-012

## Out of Scope

- Inline spreadsheet editing
