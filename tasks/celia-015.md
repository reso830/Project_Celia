# CELIA-015 - Transaction Filtering

## Epic

Transactions

## Goal

Allow users to quickly locate transactions.

## User Story

As a user,
I want to filter and search my transactions
so that I can quickly find specific entries.

## Scope

Support:

- Search
- Member filter
- Income / Expense filter
- Bucket filter
- Date range filter

## Acceptance Criteria

- [ ] Search works
- [ ] Filters work independently
- [ ] Filters combine correctly
- [ ] Empty search state shown
- [ ] Tests added
- [ ] Verification pipeline passes
